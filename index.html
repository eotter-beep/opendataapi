About the Project

OpenDataAPI is an open-source initiative that provides easy and transparent access to scraped public data. The goal is to make information more usable while respecting community concerns. Use our data at api/api.json!

Contribute

Developers, researchers, and data enthusiasts are welcome to contribute! Check out our GitHub repository to submit pull requests, raise issues, or improve documentation.
